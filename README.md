## 🧑‍💻 About This Project

variational-learning-playground was created as a simple tool to explore modern machine learning methods that focus on uncertainty and approximate inference. These methods help machines learn from data by estimating possible errors and unknowns.

The playground provides a way to experiment without deep coding knowledge. It supports a wide range of variational learning topics including deep learning and language models built with Python and PyTorch.
